Evaluación Final Transversal – Desarrollo Orientado a Objetos I

Este proyecto corresponde a la segunda entrega sumativa del caso empresarial Salmontt para la asignatura Desarrollo Orientado a Objetos I. El sistema está desarrollado en Java e implementa los principios de la programación orientada a objetos (encapsulamiento, herencia, polimorfismo, interfaces y composición). El objetivo es gestionar empleados de una empresa salmonera, utilizando lectura de archivos, validaciones y operaciones de búsqueda.

El proyecto contiene las siguientes áreas principales: 
- app/: clase principal del sistema  
- model/: clases de dominio  
- service/: lógica de negocio  
- util/: validadores y utilidades
